# FrontEnd KIET FSD 🚀

Welcome to my **Full Stack Development (FSD)** repository! This project is a comprehensive collection of frontend lab work, assignments, and practice modules completed as part of the curriculum at KIET.

## 📂 Project Structure

This repository contains multiple standalone React applications and practice modules, each focusing on a specific frontend concept:

| Module | Description |
| :--- | :--- |
| **[reduxDemo](./reduxDemo)** | State management implementation using **Redux**. |
| **[context-api](./context-api)** | Handling global state using the **React Context API**. |
| **[Routing](./Routing)** | Single Page Application (SPA) navigation with dynamic routes and Detail components. |
| **[EcomApp](./EcomApp)** | A mini e-commerce interface featuring product listings. |
| **[News](./News)** | A dynamic news application fetching real-time data from APIs. |
| **[MacroScope](./MacroScope)** | UI components styled using **Tailwind CSS**. |
| **[API](./API)** | Practice with asynchronous JavaScript, Promises, and `fetch`. |
| **[Knights](./Knights)** / **[lumen](./lumen)** | Specialized UI layouts and themed components. |

## 🛠️ Tech Stack

* **Core:** JavaScript (ES6+), HTML5, CSS3
* **Library:** [React.js](https://reactjs.org/)
* **State Management:** Redux, Context API
* **Styling:** Tailwind CSS, CSS Modules
* **Routing:** React Router DOM

## 🚀 Getting Started

To run any of the projects locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/Vishal795-knightrider/FrontEnd_KIET_FSD.git](https://github.com/Vishal795-knightrider/FrontEnd_KIET_FSD.git)
    ```
2.  **Navigate to the desired project folder:**
    ```bash
    cd reduxDemo  # Replace with the folder you want to explore
    ```
3.  **Install dependencies:**
    ```bash
    npm install
    ```
4.  **Start the development server:**
    ```bash
    npm start
    ```
